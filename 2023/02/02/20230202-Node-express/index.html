Express 搭建 web

首先安装依赖
以及一些重要模块
body-parser : 解析 http 请求体, 用于处理 JSON, Raw, Text 和 URL 编码的数据。
cookie-parser : 解析 Cookie 的工具, 方便操作客户端中的 cookie 值
multer : 处理 multipart/form-data 类型的表单数据

新建文件 创建服务

路由就是通过 get,post 第一个参数匹配，且可用正则匹配

express-generator

或者直接使用 Express 应用程序生成器 express-generator

命令会直接在当前目录下生成应用程序 默认是 jade 模板引擎

也可以通过 express –view=ejs|hbs|hjs|jade|pug|twig|vash 修改

生成完成进入项目目录 npm install 安装依赖

安装完成 npm start 启动

默认启动 3000 端口,访问 localhost:3000
